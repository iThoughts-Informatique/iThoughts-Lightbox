A flexible, responsive and customizable lightbox plugin for WordPress. Express the beauty of your high definition images!

== Description ==

This plugin allow you to create lightboxes with following functionnalities:

* select the images you want to include/exclude from the lightbox with the TinyMCE new Lightbox button
* create lightboxes with title and caption
* zoom on images that can be displayed bigger with the JS library [ImageZoom](https://github.com/Nihilivin/ImageZoom)
* merge with [iThoughts Tooltip Glossary](https://wordpress.org/plugins/ithoughts-tooltip-glossary/), and allow to include into lightboxes the mediatips you select
* create easily new transitions, animations or theme with the incoming patterns available at [GerkinDevelopment official page](http://www.gerkindevelopment.net/en/portfolio/ithoughts-lightbox/)

For more informations, please visit the [dedicated plugin page](http://www.gerkindevelopment.net/en/portfolio/ithoughts-lightbox/)

== Installation ==

Either install through admin panel (this is the eaiest way), or:

1. Upload `ithoughts-lightbox.zip` to the `/wp-content/plugins/` directory
2. Unzip the archive
3. Activate the plugin through the 'Plugins' menu in WordPress

Options will be available into `Settings` > `iThoughts Lightbox`
